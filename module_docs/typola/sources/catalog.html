typola.sources.catalog

Catalog of known typology data sources.

Only includes sources with stable public download URLs and permissive-enough licensing for research use. Users can register more sources with register_source(SourceSpec(…)).

Additional CLDF datasets worth considering (not auto-registered here, but the same loader should work once you pass a download URL):

  • APiCS (Atlas of Pidgin and Creole Language Structures)

  • SAILS (South American Indigenous Language Structures)

  • PHOIBLE (phonological inventories)

  • Lexibank cross-linguistic lexical databases
