typola.sources.base

Source specification & registry.

A source is a handle for one typology dataset — enough metadata to download it, cite it, and load it, but no data.

class typola.sources.base.SourceSpec(name: str, url: str, citation: str = '', license: str = '', archive_type: str = 'auto', strip_components: int = 0, loader: Callable | None = None)[source]

Describe a typology dataset that can be loaded into a Typology.

name

Short identifier used everywhere else ("wals", "grambank", …).

Type:

str

url

Direct download URL for a zip/tarball.

Type:

str

citation

Citation string to include with derived outputs.

Type:

str

license

License of the dataset (e.g. “CC-BY-NC-4.0”).

Type:

str

archive_type

"zip" or "tar.gz". Auto-detected from URL if left as "auto".

Type:

str

strip_components

Top-level directory entries to strip after extraction (for archives that wrap everything in a single dir named after the release).

Type:

int

download(*, force: bool = False, verbose: bool = True) → Path[source]

Download & extract the source archive. Returns the extraction root.
